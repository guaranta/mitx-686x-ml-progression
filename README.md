# mitx-686x-ml-progression

**MITx 6.86x — Machine Learning with Python: From Linear Models to Deep Learning**

Repositório de portfólio com três módulos que reproduzem a progressão do MicroMasters em Statistics and Data Science: classificação linear, visão computacional e aprendizado não supervisionado.

| Módulo | Tópico | Comando |
|--------|--------|---------|
| `01_sentiment` | Perceptron, Average Perceptron, Pegasos | `python 01_sentiment/run.py` |
| `02_mnist` | Linear → MLP no MNIST (subset) | `python 02_mnist/run.py` |
| `03_matrix_completion` | EM + mistura gaussiana (Netflix-style) | `python 03_matrix_completion/run.py` |

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
